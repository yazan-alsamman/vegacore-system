## Color Palette

| Name | Hex | Usage |
|------|-----|--------|
| Navy | `#2E3192` | Primary brand, sidebar, headings |
| Cyan | `#00AEEF` | Accents, links, active states |
| Black | `#231F20` | Body text |
| Red | `#ED1C24` | Alerts, urgent, logout |
| Green | `#00A651` | Success, active status |
| Magenta | `#EC008C` | Secondary accent |
| Yellow | `#FFF200` | Highlights |
| Gold | `#F7B040` | Warnings |

## Gradients

- **Primary:** `#2E3192` → `#00AEEF`
- **Hero:** Navy depth with cyan glow
- **Accent bar:** RGB spectrum (red → magenta → navy → cyan → green)

## Typography

- **Font:** Plus Jakarta Sans (Google Fonts)
- **Wordmark:** VEGA**CORE** (CORE in cyan)

## UI Patterns

- Hexagonal honeycomb overlay on hero/sidebar
- Dark navy sidebar with light navigation
- Gradient CTA buttons
- Brand-colored chart series
