# VegaCore OS – Mobile App (Flutter)

Planned Flutter mobile client consuming the same REST API (`/api/v1`).

## Planned features

- JWT authentication with secure token storage
- Dashboard KPIs
- Task management
- Notifications push
- RTL Arabic support

## API Base URL

Configure `API_BASE_URL` to your VegaCore API instance (e.g. `https://yourdomain.com/api/v1`).

## Status

Scaffold pending – backend and web clients are production-ready. Mobile will share the OpenAPI spec from `/api/docs`.
